# Basic "AND" Gate Perceptron 
###### Andre Lennard S. Uy

This is a simple perceptron that implements the AND gate. The perceptron is trained using the perceptron learning rule. The perceptron is then tested using the test data.

**Inputs**   
1 , 1  
1 , 0  
0 , 1  
0 , 0

**Expected Output**  
1  
0  
0  
0
